Social App - React.js  

Social App is an interactive web application similar to social media platforms, built with React.js. It allows users to manage their content by creating, editing, and deleting posts and comments, with the ability to customize their profile picture. The application is fully responsive across all devices.  

[Live Demo](https://social-app-lake-nine.vercel.app/)

✨ Features

· Post Management: Add, view, edit, and delete posts  
· Comment System: Add and delete comments on posts  
· User Profile: Change and update profile picture  
· Responsive Interface: Fully responsive design that works on all screen sizes (mobile, tablet, desktop)  
· Interactive State: Using useState and useEffect for dynamic state management  

🛠️ Technologies Used  

· Frontend Framework: React.js  
· Routing: React Router DOM (for navigation between pages)  
· Styling: CSS Modules / Tailwind CSS / CSS / Hero ui

🚀 Getting Started  

Follow these steps to run the project locally:  

Prerequisites

· Node.js (version 14 or higher)
· npm

Installation

1. Clone the repository
   bash
   git clone https://github.com/omarkhattab74/social-app.git
    
2. Install dependencies
   bash
   npm install
   
4. Run the development server
   bash
   npm start run dev
   
5. Open your browser
   The app will automatically open at http://localhost:3000.
